### 新建HAL库版本MDK工程

##### 准备工作

> 下载官方固件包（ST官网STM32Cube）

##### 新建步骤

1. 新建工程文件夹：D/M/O/P/U五个文件夹
   | 文件夹名称  | 作用                                                          |
   | ----------- | ------------------------------------------------------------- |
   | Drivers     | 存放与硬件相关的文件                                          |
   | Middlewares | 存放正点原子提供的中间层组件文件和第三方中间层文件            |
   | Output      | 存放工程编译输出文件                                          |
   | Projects    | 存放 MDK 工程文件                                             |
   | User        | 存放HAL库用户配置文件、main.c、中断处理文件，以及分散加载文件 |
2. 新建工程框架
   工程命名，选芯片 
3. 添加文件
   1. 设置工程名和分组名
   2. 添加.s, .c
4. 魔术棒设置
   T/O/L/C/D/U,六个选项卡 
5. 添加main.c
